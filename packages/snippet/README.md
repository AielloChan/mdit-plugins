snippet plugin for MarkdownIt.

## Install / 安装

```bash
# pnpm
pnpm add -D @mdit/plugin-snippet
# npm
npm i -D @mdit/plugin-snippet
# yarn
yarn add -D @mdit/plugin-snippet
```
